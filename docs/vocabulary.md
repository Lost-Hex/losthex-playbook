# Vocabulary

Abbreviations and technical terms used throughout this documentation.

---

## How to Use

1. Add terms to the tables below
2. Run `npm run build:vocab` to generate `tooling/vale-styles/config/vocabularies/LostHex/accept.txt`
3. Vale will stop flagging these terms as unknown

**Table format:**

| Term | Description |
|------|-------------|
| API | Application Programming Interface |
| AWS, S3, RDS | Amazon Web Services components |

- First column: term (or comma-separated aliases)
- Second column: description (for human readers)
